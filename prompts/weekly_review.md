# Weekly Review Prompt

週1回、日本語ユーモア記事メディアの反応分析をしてください。

## 入力

- `performance.csv`
- `reader_reviews.csv`
- `compliance_reviews.csv`
- `publishing_queue.csv`
- 公開済み記事の一覧
- SNS反応、コメント、検索流入、読了率

## 手順

1. PV、読了率、平均滞在時間、SNS共有数で記事を分類する。
2. 伸びた記事の共通点を抽出する。
3. 伸びなかった記事の共通点を抽出する。
4. readerスコア、実際の反応、公開後の読了率のズレを見る。
5. 辛口読者が低く見たのに伸びた記事、辛口読者が高く見たのに伸びなかった記事を分ける。
6. 炎上兆候、誤解、権利リスク、広告表記リスクがあった記事を記録する。
7. 次週の企画ルールを更新する。

## 出力

- 今週の勝ちパターン
- 今週の負けパターン
- 使い回してよい型
- 使いすぎ注意の型
- NGルール追加案
- 次週の企画テーマ
- `article_editor_in_chief` と3人のライターに渡す更新済み指示
